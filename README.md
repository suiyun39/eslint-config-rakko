# eslint-config-rakko

自用 ESLint 规则集

## 安装与使用

```shell
pnpm add -D eslint eslint-config-rakko
```

**eslint.config.js**

```javascript
import { defineConfig } from 'eslint-config-rakko'

export default defineConfig({
  react: true,
  typescript: true,
})
```
